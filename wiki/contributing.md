# Contributing

### Contributing to the Wiki

We want this wiki to be a community-driven project! If you’ve discovered a rare spawn location or written a guide for a complex gym battle, we’d love your help.

#### How You Can Help

1. Update Spawn Data: Found a Pokémon in a biome we haven't listed? Let us know.
2. Fix Typos: Keep our pages looking professional.
3. Write Guides: Share your expertise on breeding, EV training, or base building.

#### Submission Guidelines

* Be Accurate: Please verify information in-game before posting.
* Stay Relevant: Only include mods and features currently active on the server.
* Respect Others: Use clear, helpful language and follow the server's code of conduct.

_To request "Editor" status, please contact a Moderator on Discord._
