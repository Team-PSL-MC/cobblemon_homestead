# 🤝 Contributing

Cobblemon Homestead is a **community-driven project**.  
Players can help improve the server by contributing to the **wiki, spawn data, quests, builds, and gameplay systems**.

If you enjoy documenting mechanics, designing challenges, or improving balance, we'd love your help.

---

# 📝 Wiki Contributions

The wiki helps players understand the server's systems and mechanics.

### Ways You Can Help

- Fix typos or formatting issues
- Improve existing guides
- Write new guides for mechanics or mods
- Add helpful screenshots
- Update outdated information

Examples of useful guides:

- Breeding strategies
- EV training
- Base building tips
- Factory automation
- Gym challenge strategies

### Guidelines

Please follow these rules when editing the wiki:

- **Be Accurate** – Verify information in-game before posting.
- **Stay Relevant** – Only document features active on the server.
- **Be Clear** – Use simple and helpful explanations.
- **Respect Others** – Follow the server's code of conduct.

To request **Wiki Editor access**, contact a **Moderator on Discord**.

# 🧬 Pokémon Spawn Contributions

All Pokémon spawn data is managed in our GitHub repository.

Players can suggest improvements or submit pull requests if they discover better spawn locations or balance ideas.

### Spawn Data Repository

https://github.com/Team-PSL-MC/cobblemon_homestead/tree/main/data/cobblemon/spawn_pool_world

### What You Can Contribute

Examples:

- New biome spawn suggestions
- Spawn rate balancing
- Time-of-day spawn improvements
- Rare Pokémon adjustments

### Guidelines

- Test changes **in-game if possible**
- Avoid excessive spawn rates
- Keep biome themes logical

Pull requests and issues are welcome.
